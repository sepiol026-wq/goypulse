## О модуле
Recon — технический модуль для recon/OSINT-задач: сбор базовой сетевой и доменной информации для анализа инфраструктуры.

## Как это работает
Модуль запускает набор проверок (DNS/хост/сетевые признаки) и возвращает результат в чат в удобном формате для первичного анализа.
