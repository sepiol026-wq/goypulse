## What this module is
GoyVirus is a humorous prank module for playful chats with friends. It imitates a “virus-like” joke scenario and is not meant for harmful activity.

## ⚠️ Important warning
- Use it only for harmless entertainment in appropriate chats.
- Do not run it in work/school/sensitive contexts.
- The author **is not responsible** for misuse or resulting consequences.

## Module file
- `goyv.py`

## Installation
```text
.dlm https://raw.githubusercontent.com/sepiol026-wq/GoyModules/main/goyv.py
```

## Usage
Load the module and run prank commands in a private test chat with friends. The behavior is comedic and does not provide real malware functionality.

## Commands
- The module ships with prank-oriented commands (check in-module help after loading).
