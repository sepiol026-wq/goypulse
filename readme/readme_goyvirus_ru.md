## Что это за модуль
GoyVirus — шуточный тролль-модуль для розыгрышей друзей в Telegram. Он имитирует «вирусный» сценарий в формате прикола и не предназначен для вредоносной активности.

## ⚠️ Важное предупреждение
- Используйте модуль только в юмористических целях и только там, где это уместно.
- Не отправляйте в рабочие, учебные и чувствительные чаты.
- Автор **не несёт ответственности** за последствия неправильного или токсичного использования.

## Файл модуля
- `goyv.py`

## Установка
```text
.dlm https://raw.githubusercontent.com/sepiol026-wq/goypulse/main/goyv.py
```

## Как использовать
После установки активируйте команды из модуля в тестовом чате с друзьями. Формат — «пранк-диалог», без реального заражения или доступа к данным.
