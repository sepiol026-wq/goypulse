## Что это за модуль
GoyPulse V9 — продвинутый нейро-автоответчик на основе цепей Маркова. Модуль учится на сообщениях, учитывает контекст чата, поддерживает скрытый режим и систему резервного копирования.

## Как это работает
1. Считывает историю и новые сообщения в чатах.
2. Обновляет языковую модель по локальной памяти.
3. Генерирует ответ с учётом настроек, ограничений и режима работы.
4. Позволяет бэкапить/восстанавливать состояние.

## Файл модуля
- `goypulse.py`

## Установка
```text
.dlm https://raw.githubusercontent.com/sepiol026-wq/goypulse/main/goypulse.py
```

## Основные команды
- `.gpulse on/off` — включение и отключение.
- `.gpref` — обновить память/рефреш.
- `.gpstat` — статус и статистика.
- `.gpinfo` — диагностика текущего чата.
- `.gpupdate` — проверка/применение обновлений.
- `.gpbackup` — операции резервного копирования.

## Рекомендованный старт
1. Загрузите модуль.
2. Включите `.gpulse on` в нужном чате.
3. Через время выполните `.gpref` для улучшения ответов.
4. Настройте бэкапы через `.gpbackup`.
