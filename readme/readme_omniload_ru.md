## О модуле
OmniLoad — универсальный загрузчик медиа, который позволяет быстро скачать контент по ссылке и переотправить его прямо из чата.

## Как работает
Модуль принимает ссылку/вложение, определяет источник, загружает медиа и отправляет результат в текущий диалог.

## Файл
- `omni.py`

## Установка
```text
.dlm https://raw.githubusercontent.com/sepiol026-wq/goypulse/main/omni.py
```

## Команды
- `.dl` — базовая команда загрузки.
