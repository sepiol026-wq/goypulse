## О модуле
Спизди все ии ключи в любом чате.

## Как это работает
Модуль сканирует историю чата по набору префиксов и regex-шаблонам, валидирует найденные ключи по API провайдеров и сохраняет уникальные результаты в БД. Есть авто-режим для чата, глобальный скан, экспорт/импорт и статистика по провайдерам.

## Файл

## Установка

## Команды
- Модуль содержит набор команд/inline-кнопок для сканирования, проверки, экспорта, импорта и управления базой ключей. Точный список смотрите через help внутри бота после загрузки.
