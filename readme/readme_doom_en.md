## About module
Doom is an entertainment module that launches an inline mini-game directly inside Telegram.

## Usage flow
1. Install the module.
2. Run `.doom` to open game menu.
3. Use `.hdoom` for quick help.

## Module file
- `doom.py`

## Installation
```text
.dlm https://raw.githubusercontent.com/sepiol026-wq/GoyModules/main/doom.py
```

## Commands
- `.doom`
- `.hdoom`
