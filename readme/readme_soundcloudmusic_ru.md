## О модуле
SoundCloudMusic превращает ваш userbot в музыкальный инструмент: поиск треков, выгрузка в чат и управление локальными базами-плейлистами.

## Как это работает
- Ищет треки в SoundCloud.
- Скачивает/отправляет найденный трек.
- Позволяет хранить персональные наборы треков в локальной БД.
- Поддерживает импорт/экспорт и пакетное наполнение базы.
